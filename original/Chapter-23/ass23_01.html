CHAPTER 23 — Logic and Math Pseudoinstructions

This chapter discusses additional pseudoinstructions. When you write a program using pseudoinstructions it looks to you as if the computer has a rich set of convenient machine instructions.

In fact, these convenient instructions are each implemented by using basic instructions.

The extended assembler replaces each pseudoinstruction with one or more basic instructions. The basic instructions are then translated into machine code.
