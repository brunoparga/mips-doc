To print a floating point value to the SPIM monitor, use service 2 (for single precision) or service 3 (for double precision).

To read a floating point value from the user, use service 6 (for single precision) or service 7 (for double precision). These notes deal mostly with single precision.

The chart shows the list of SPIM exception handler services. Each I/O method works with a specific format for data. The methods for double use an even-odd pair of registers.
