Example Program

Let us write a program that uses the Simple Linkage convention. The program reads three integers from the user and computes the sum. The outline of the program is: 

# read first integer

# read second integer

# read third integer

# compute the sum

# write out the result

Of course, the user will enter integers as characters from the keyboard. The program uses the exception handler service number five to read the characters and convert them into a 32-bit integer.
