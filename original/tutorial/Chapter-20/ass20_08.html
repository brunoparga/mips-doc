Example Program: sum of an array

Sometimes programmers try to end an array with a pattern that they think will never occur in the data, like 0x00000000, or 0xFFFFFFFF. This is an invitation for bugs. The pattern might truly never occur in correct data, but faulty data is common, and your program should deal with it gracefully.
