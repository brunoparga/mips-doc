What is the mantissa when the entire float is zero?

Answer:

Zero.


Exponent Practice

The sign bit is zero when the number is zero. A floating point zero consists of 32 zero bits. The MIPS instructions that test for integer zero can test for floating point zero.

IEEE floating point also represents "negative zero", where the sign bit is set (to one), and all other bits are zero. In mathematics, negative zero is the same as zero. But floating point calculations sometimes give different results for "positive zero" and "negative zero". For these chapters assume that zero is represented with 32 zero bits.
