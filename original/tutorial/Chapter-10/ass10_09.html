Each MIPS instruction is 32 bits (four bytes) long. The PC is incremented to point at the next instruction.


Sequential Execution

MIPS machine instructions are all 32 bits wide (4 bytes). Normally, instructions are executed one after another starting with the first instruction of the program and proceeding in sequence through memory. The execution sequence can be changed with a branch or a jump machine instruction.
