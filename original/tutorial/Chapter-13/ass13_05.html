Answer:

Both operand registers and the result register can all be the same. (This is not true on many processors, but MIPS is nice in this way.)

The effect of adding an integer to itself is doubling it. The maximum unsigned integer that can be represented in 32 bits is +232-1 so the maximum that $8 could start with is: +231-1


Example Program

## AddSome.asm
## 
## Program to demonstrate addition
        .text
        .globl  main

main:
        ori      $8, $0, 0xAB       # put 0x000000AB into $8
        ori      $9, $0, 0x55       # put 0x00000055 into $9
        addu     $10,$9, $8         # $10  gets sum

## End of file
