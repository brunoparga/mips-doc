The la is the correct one to use because we need the address that print stands for, not the contents of that location.


Calling the Methods

After constructing the object, the program invokes each method of the object. 

     
                                      # object1.read();
          lw       $a0,          #   get address of object1
          
          lw       $t0, ($a0)    #   get address of read method
          
          jalr                   #   call the method
          
                                      # object1.print();
          lw       $a0,          #   get address of first object
          
          lw       $t0, ($a0)    #   get address of print method
          
          jalr                   #   call the method
          
          li      $v0,10              # return to OS
          syscall
